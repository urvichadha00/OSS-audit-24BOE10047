## Description
This project is an audit of the open source software python. It includes a detailed report and five shell scripts.

## Scripts

script1.sh shows system information like user, kernel, uptime and date  

script2.sh checks whether python is installed on the system  

script3.sh analyzes important system directories and shows permissions and size  

script4.sh reads a log file and counts how many times a keyword appears  

script5.sh generates a simple open source manifesto based on user input
